## Фаза 3 — карусель фокусных точек на главной

Единственное, что осталось из исходного трёхэтапного плана: заменить статичный блок фокусных точек на главной странице горизонтальной каруселью.

### Что делаем

1. **Карусель вместо статичной сетки**
   - В `NormPrototype.tsx` заменить текущий блок `HOME_FOCUS_POINTS` (три плитки в ряд) на горизонтально прокручиваемую ленту.
   - Источник данных — весь массив `FOCUS_POINTS` (канонический порядок), но на главной сохранить приоритет видимости `["fp-delivery", "fp-supply", "fp-it"]` первыми (заморозка визуального порядка из пункта 5 согласованного плана).
   - Клик по карточке открывает существующий `FocusPointModal` — логику модалки не трогаем.

2. **Поведение и навигация**
   - Прокрутка колесом/трекпадом + drag-to-scroll (native `overflow-x: auto` со scroll-snap по карточкам).
   - Стрелки влево/вправо (появляются при наличии скрытых карточек, скрываются в крайних положениях).
   - Клавиатура: `←/→` для перелистывания, `Tab` по карточкам, `Enter` открывает модалку.
   - На мобильной ширине (текущий viewport 360px) — по одной карточке в поле зрения со snap.

3. **Оформление**
   - Ширина карточки фиксированная (≈320px десктоп, 88vw мобилка), высота выравнивается по самой высокой.
   - Скрыть системный скроллбар, добавить мягкие градиентные «затухания» слева/справа как индикатор прокрутки.
   - Никаких изменений внутренней вёрстки карточки — только контейнер.

4. **Что НЕ трогаем**
   - Порядок остальных секций главной.
   - `FOCUS_POINTS` и `SOURCES_INDEX` (уже канонические после Фазы 1).
   - `FocusPointModal`, `CompanySummaryModal`, `SourceDrawer`, `RiskDetailModal`.
   - Стили других блоков.

### Технические детали

- Новый компонент `FocusCarousel` внутри `NormPrototype.tsx` (или отдельный файл `src/components/FocusCarousel.tsx`, если разрастётся) — принимает `points: FocusPoint[]` и `onOpen(id)`.
- Реализация без сторонних библиотек: `useRef` на контейнер, `scrollBy({ left, behavior: "smooth" })` для стрелок, `IntersectionObserver` или `scroll` listener для состояния стрелок.
- Новые CSS-классы в `src/styles/norm-prototype.css`: `.np-focus-carousel`, `.np-focus-carousel-track`, `.np-focus-carousel-card`, `.np-focus-carousel-nav`, `.np-focus-carousel-fade`.
- Порядок для главной: `[...HOME_FOCUS_POINTS, ...FOCUS_POINTS.filter(p => !HOME_FOCUS_POINTS.includes(p))]` — с дедупликацией по `id`.

### Проверка

- TypeScript чистый.
- Playwright: скриншот главной на 1280px и 360px, клик по первой и последней карточке — открывается корректный `FocusPointModal`.
- Escape закрывает модалку, карусель сохраняет позицию скролла.
